## AI 工作指南

当你开始工作时:

1. **读取本文档** → 了解当前进度
2. **读取当前阶段文档** → 找到下一个待做任务
3. **执行任务** → 完成后更新任务状态
4. **更新本文档** → 同步进度条和完成数

### 状态更新规则

```markdown
任务完成时:
1. 将任务状态从 ⬜ 改为 ✅
2. 更新阶段文档的验收标准 checkbox
3. 更新本文档的进度条和完成数
4. 如果阶段完成，将阶段状态改为 ✅ 已完成
```
